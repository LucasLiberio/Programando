Tabelas

Hierarquia de tabelas (simples)

  • Table = tabela
    1. caption
    2. colgroup
      • col
      3. = serve para seleçao de uma coluna inteira atraves de class atraves do col| cmd span funciona como expançao de coluna
    3. thead
      • tr
        • th
    4. tbody
      • tr
        • td
    5. tfoot
      • tr
        • th
    • table row = linha de tabela = tr
      • legenda/ titulo = caption

      • table header = cabeçalho de tabela = th

        1. cmd scope="col" representa q esse titulo é pra coluna onde ele esta

        2. cmd scope="colgroup" esse titulo é para o grupo de colunas q ele esta representando

      • table data = dado de tabela = td
        1. *cmd colspan = expasao de coluna 'na mesma linha'

        2. *cmd rowspan = expasao de linha 'na mesma coluna'

      • titulo na tabela = th
        cmd scope="row" representa q esse titulo é para a linha q ele esta
        cmd scope="rowgroup" esse titulo é para o grupo de linhas q ele esta representando
